# National Visual Direction

Use this direction if a future Figma screen or asset is needed for Himma AI:

- keep the current Himma website identity as the source of truth
- use black, Jordanian green, red, white, and restrained gold accents
- keep the logo of the main website unchanged
- use a compact AI mark for the floating assistant, not a replacement initiative logo
- avoid decorative clutter and keep the assistant UI practical, readable, and mobile-friendly
- preserve RTL-first layout and provide English text for translated UI

Current AI mark:

- circular white badge
- stylized "AI" letterform
- subtle red, green, and gold signal dots
- embedded as SVG through JavaScript/CSS, with no external image request
